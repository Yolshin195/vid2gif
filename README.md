# Video to GIF Converter CLI 🎥 Utility

Vid2GIF CLI — Простой инструмент командной строки на Python для быстрого создания анимированных GIF из видеофайлов. Инструмент равномерно извлекает заданное количество кадров по всей длительности видео, позволяя получить "краткое содержание" ролика в одном файле.

## Особенности
* **Равномерная выборка:** Не просто первые $N$ кадров, а распределенные по всему хронометражу.
* **Авто-ресайз:** Возможность ограничить ширину для контроля веса GIF.
* **Makefile:** Установка и запуск одной командой.
* **OpenCV + ImageIO:** Надежная связка для обработки видео и качественной сборки анимации.

## Быстрый старт

### 1. Требования
* Python 3.8+
* Утилита `make` (предустановлена в Linux и macOS)

### 2. Установка
Просто выполните команду в корне проекта, чтобы создать виртуальное окружение и установить зависимости:
```bash
make setup
```

### 3. Использование
Самый простой запуск (использует параметры по умолчанию):
```bash
make run IN=your_video.mp4
```

Или через прямое обращение к скрипту внутри окружения:
```bash
source venv/bin/activate
python vid2gif.py path/to/video.mp4 -o result.gif -n 30 -w 640
```

## Аргументы командной строки

| Аргумент | Короткий флаг | Описание | По умолчанию |
| :--- | :--- | :--- | :--- |
| `input` | — | Путь к исходному видео (mp4, avi, mov и т.д.) | — |
| `--output` | `-o` | Путь к сохраняемому файлу | `result.gif` |
| `--num` | `-n` | Сколько кадров извлечь из видео | `24` |
| `--width` | `-w` | Ширина выходного GIF в пикселях | (оригинал) |

## Команды Makefile
* `make setup` — создание `venv` и установка библиотек из `requirements.txt`.
* `make run` — запуск конвертации. Можно передавать переменные: `make run IN=v.mp4 OUT=res.gif N=50 W=320`.
* `make clean` — полная очистка проекта от временных файлов и виртуального окружения.
